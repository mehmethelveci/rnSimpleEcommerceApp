# Installation

```
git clone https://github.com/mehmethelveci/rnSimpleEcommerceApp.git

cd rnSimpleEcommerceApp```

npm install

```

### for ios:
```
cd ios && pod install && cd ..

react-native run-ios
```

------

### for android:
```
react-native run-android
```
